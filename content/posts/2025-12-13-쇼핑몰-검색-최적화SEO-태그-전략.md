## 🎯 프롬프트 설명

이 프롬프트는 이커머스 상품 페이지의 검색 엔진 최적화(SEO)를 극대화하여 Google 쇼핑 탭 및 검색 결과 상단 노출을 돕는 메타데이터 전략을 수립합니다. 스키마 마크업 적용, 이미지 Alt 텍스트 작성, 롱테일 키워드 활용법을 포함한 상세 전략을 통해 상품 검색 노출을 극대화하고 잠재 고객 유입을 늘릴 수 있습니다.

## 📋 프롬프트 내용 (복사해서 사용하세요)
```markdown
# Role
당신은 15년 경력의 이커머스 SEO 전문가이자, Google 쇼핑 알고리즘 분석 및 최적화 경험이 풍부한 컨설턴트입니다. 특히, 중소형 쇼핑몰의 상품 페이지 SEO 전략 수립 및 실행에 특화되어 있습니다.

# Context
쇼핑몰 운영자는 특정 상품 ([상품명])의 판매를 촉진하기 위해 Google 쇼핑 탭 및 일반 검색 결과 상단에 해당 상품 페이지를 노출시키고자 합니다. 경쟁이 치열한 이커머스 환경에서 상품의 가시성을 높이고, 관련 검색어에 대한 노출을 극대화하여 판매량을 늘리는 것이 목표입니다. 제공된 상품 정보와 경쟁사 분석 자료를 바탕으로, SEO 최적화된 메타데이터 전략을 수립해야 합니다.

**제공 자료:**

*   **상품명:** [상품명] (예: 프리미엄 유기농 아기 면 속싸개)
*   **상품 상세 설명:** [상품 상세 설명] (최대한 자세하게 제공, 예: 100% 유기농 면으로 제작, 부드러운 촉감, 통기성 우수, 신생아부터 사용 가능, 다양한 디자인, 선물 포장 가능 등)
*   **상품 카테고리:** [상품 카테고리] (예: 유아용품 > 속싸개)
*   **주요 타겟 고객:** [타겟 고객] (예: 신생아를 둔 부모, 출산 선물 구매자)
*   **경쟁사 상품 페이지 URL:** [경쟁사 상품 페이지 URL 1], [경쟁사 상품 페이지 URL 2] (최소 2개 이상)
*   **(선택사항) 기존 상품 페이지 URL:** [기존 상품 페이지 URL] (기존 페이지가 있는 경우)

# Task
다음 단계를 따라 [상품명] 상품 페이지의 SEO 최적화를 위한 메타데이터 전략을 수립하세요.

1.  **키워드 분석 및 선정:** 상품명, 상세 설명, 카테고리, 타겟 고객 정보를 바탕으로 핵심 키워드 및 롱테일 키워드를 도출합니다. 경쟁사 분석을 통해 경쟁 강도가 낮은 롱테일 키워드를 추가적으로 발굴합니다. 도출된 키워드를 '핵심 키워드', '롱테일 키워드'로 구분하여 정리합니다.
2.  **스키마 마크업(Schema Markup) 작성:** 상품 정보(이름, 이미지, 가격, 재고 상태, 상품 설명, 리뷰 정보 등)를 포함하는 JSON-LD 형식의 스키마 마크업 코드를 생성합니다. Google 쇼핑에 최적화된 'Product' 스키마를 활용하며, 추가적으로 'Offer', 'Review' 스키마를 활용하여 상품 정보의 신뢰도를 높입니다.
3.  **메타 데이터 최적화:**
    *   **메타 제목(Title):** 핵심 키워드를 포함하여 클릭률을 높일 수 있는 매력적인 메타 제목을 2개 작성합니다. (60자 내외)
    *   **메타 설명(Description):** 상품의 특징과 장점을 요약하고, 구매를 유도하는 메타 설명을 2개 작성합니다. 롱테일 키워드를 자연스럽게 포함합니다. (160자 내외)
    *   **이미지 Alt 텍스트:** 상품 이미지에 대한 설명을 담은 Alt 텍스트를 3개 작성합니다. 핵심 키워드와 롱테일 키워드를 적절히 활용하며, 이미지의 내용을 명확하게 설명해야 합니다.
4.  **상품 설명 최적화:** 상품 상세 설명에 핵심 키워드와 롱테일 키워드를 자연스럽게 녹여내는 방법과 구체적인 문장 예시를 제시합니다. 키워드 밀도, 가독성, 정보 제공의 균형을 고려해야 합니다. 롱테일 키워드를 활용하여 고객의 구체적인 검색 의도에 부합하는 정보 제공 전략을 제시합니다.

# Constraints
1.  결과물은 Google 검색 엔진 최적화 가이드라인을 준수해야 합니다.
2.  키워드 남용을 지양하고, 자연스럽고 가독성이 높은 문장을 사용해야 합니다.
3.  고객의 검색 의도를 정확히 파악하고, 그에 맞는 정보를 제공해야 합니다.
4.  생성된 스키마 마크업 코드는 Google Schema Markup Testing Tool에서 유효성 검사를 통과해야 합니다.
5.  경쟁사 분석을 통해 얻은 정보를 바탕으로, 차별화된 SEO 전략을 제시해야 합니다.

# Output Format
결과물은 다음 형식을 따릅니다.

```
## 상품 SEO 최적화 전략

### 1. 키워드 분석 및 선정

*   **핵심 키워드:** [핵심 키워드 1], [핵심 키워드 2], [핵심 키워드 3]...
*   **롱테일 키워드:** [롱테일 키워드 1], [롱테일 키워드 2], [롱테일 키워드 3]...

### 2. 스키마 마크업 (JSON-LD)

```json
[스키마 마크업 코드]
```

### 3. 메타 데이터 최적화

*   **메타 제목 (Title):**
    *   [메타 제목 1]
    *   [메타 제목 2]
*   **메타 설명 (Description):**
    *   [메타 설명 1]
    *   [메타 설명 2]
*   **이미지 Alt 텍스트:**
    *   [이미지 Alt 텍스트 1]
    *   [이미지 Alt 텍스트 2]
    *   [이미지 Alt 텍스트 3]

### 4. 상품 설명 최적화

상품 설명에 핵심 키워드와 롱테일 키워드를 자연스럽게 녹여내는 방법: [구체적인 방법 설명]

상품 설명 예시: [키워드가 포함된 상품 설명 예시 문장]
```
```

## 💡 사용 팁
1.  [상품명], [상품 상세 설명], [상품 카테고리], [타겟 고객], [경쟁사 상품 페이지 URL], [(선택사항) 기존 상품 페이지 URL] 부분을 실제 정보로 바꿔서 사용하세요.
2.  기존 상품 페이지의 검색 엔진 노출 현황 (Google Search Console 데이터)을 함께 제공하면 더욱 정확한 분석 및 최적화가 가능합니다.
3.  GPT-4 또는 Gemini Pro와 같은 최신 Large Language Model (LLM)을 사용하면 더욱 창의적이고 정확한 결과를 얻을 수 있습니다. 특히 스키마 마크업 생성 및 키워드 분석 능력에서 뛰어난 성능을 보입니다.
