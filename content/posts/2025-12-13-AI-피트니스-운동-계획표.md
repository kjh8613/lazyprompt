## 🎯 프롬프트 설명

이 프롬프트는 사용자의 운동 목표와 수준에 최적화된 개인 맞춤형 AI 피트니스 운동 계획표를 생성합니다. 일주일간의 운동 루틴을 표 형태로 제공하며, 각 운동 종목, 세트 수, 횟수, 휴식 시간 등을 상세하게 안내합니다.

## 📋 프롬프트 내용 (복사해서 사용하세요)

```markdown
**[개인 맞춤형 AI 피트니스 운동 계획표 생성]**

당신은 최고의 AI 피트니스 코치입니다. 사용자의 정보를 바탕으로 최적의 운동 계획표를 생성해주세요.

**사용자 정보:**

*   **목표:** (예: 다이어트, 근육 증가, 근력 향상, 유연성 향상, 전신 균형 발달. 구체적으로 작성)
*   **운동 수준:** (예: 초급 - 운동 경험 거의 없음, 중급 - 주 2-3회 규칙적인 운동, 고급 - 주 4회 이상 고강도 운동. 구체적으로 작성)
*   **운동 가능 시간:** (예: 주당 3일, 각 1시간)
*   **선호하는 운동 스타일:** (예: 웨이트 트레이닝, 맨몸 운동, 유산소 운동, 필라테스, 요가, 크로스핏 등. 구체적으로 작성)
*   **특별히 집중하고 싶은 부위:** (예: 복근, 하체, 상체, 힙. 없다면 "없음"으로 기재)
*   **가지고 있는 운동 장비:** (예: 덤벨, 밴드, 풀업바, 요가매트. 없다면 "없음"으로 기재)
*   **기존 운동 경험:** (구체적으로 작성. 예를 들어 "3개월간 헬스 경험, 스쿼트 50kg 5회 가능")
*   **기존 부상 경험:** (구체적으로 작성. 없다면 "없음"으로 기재)

**계획표 형식:**

아래 표 형식을 준수하여 일주일 운동 루틴을 생성해주세요. 각 요일별 운동 종목, 세트 수, 횟수, 휴식 시간을 명확하게 기재해야 합니다. 운동 종목은 구체적인 명칭 (예: 바벨 스쿼트, 덤벨 벤치프레스)으로 작성하고, 필요에 따라 운동 방법 팁 또는 주의사항을 짧게 추가하세요.

| 요일 | 운동 종목 | 세트 수 | 횟수 | 휴식 시간 | 추가 설명/주의사항 |
|---|---|---|---|---|---|
| 월요일 |  |  |  |  |  |
| 화요일 |  |  |  |  |  |
| 수요일 |  |  |  |  |  |
| 목요일 | 휴식 | - | - | - | - |
| 금요일 |  |  |  |  |  |
| 토요일 |  |  |  |  |  |
| 일요일 | 휴식 | - | - | - | - |

**고려 사항:**

*   운동 수준에 맞는 적절한 강도와 난이도로 계획을 구성하세요.
*   목표 달성을 위한 최적의 운동 조합을 고려하세요 (예: 다이어트를 위한 유산소 운동 + 근력 운동).
*   운동 부위의 균형 발달을 고려하여 계획을 구성하세요.
*   충분한 휴식 시간을 포함하여 부상 위험을 최소화하세요.
*   운동 전후 스트레칭 또는 웜업/쿨다운 루틴을 포함하는 것을 추천합니다.

**출력:**

위에서 제시된 표 형식으로 일주일 운동 계획표를 상세하게 작성하여 제공해주세요. 추가적으로, 해당 운동 계획을 통해 기대할 수 있는 효과와 운동 시 주의해야 할 점을 간략하게 요약해주세요.
```

## 💡 사용 팁

1.  **사용자 정보 상세화:** 목표, 운동 수준, 선호하는 운동 스타일 등을 최대한 구체적으로 작성할수록 AI가 더욱 정확하고 맞춤화된 계획을 생성합니다.
2.  **장비 유무 명시:** 가지고 있는 운동 장비를 명확히 기재하여, 집에서도 효율적으로 운동할 수 있는 계획을 받을 수 있습니다.
3.  **피드백 반영:** 생성된 계획을 직접 수행해보고, 힘들거나 불편한 점이 있다면 AI에게 다시 피드백하여 계획을 수정하도록 요청하세요. (예: "스쿼트 횟수가 너무 많아요. 횟수를 줄여주세요." 또는 "덤벨 벤치프레스 대신 푸쉬업으로 변경해주세요.")
