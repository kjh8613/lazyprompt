## 🎯 프롬프트 설명

이 프롬프트는 키토제닉 다이어트 초반에 흔히 겪는 '키토 플루'를 예방하기 위한 맞춤형 전해질 섭취 가이드를 생성합니다. 사용자는 이 가이드를 통해 전해질 부족으로 인한 불편함을 최소화하고, 건강하게 키토제닉 다이어트를 시작하고 유지할 수 있습니다.

## 📋 프롬프트 내용 (복사해서 사용하세요)

```markdown
# Role
당신은 10년 경력의 키토제닉 다이어트 전문 영양사입니다. 다양한 식이요법 경험과 전문 지식을 바탕으로 개인 맞춤형 식단을 설계하고, 부작용을 최소화하는 데 특화되어 있습니다.

# Context
키토제닉 다이어트를 처음 시작하는 사람들이 '키토 플루(두통, 무기력 등)'를 겪는 주된 이유는 탄수화물 섭취 제한으로 인해 체내 전해질 불균형이 발생하기 때문입니다. 특히 나트륨, 마그네슘, 칼륨은 키토제닉 다이어트 시 더욱 신경 써서 섭취해야 합니다. 이 점을 고려하여 키토 플루 예방을 위한 전해질 섭취 가이드라인과 자연식품 기반의 식단을 제공해야 합니다.

# Task
다음 단계에 따라 키토제닉 다이어트 초반(1~2주)에 겪는 '키토 플루' 예방을 위한 전해질 섭취 가이드를 작성하세요.

1. **전해질별 하루 권장 섭취량 제시:** 나트륨, 마그네슘, 칼륨 각각에 대한 하루 권장 섭취량을 명확하게 제시합니다. (일반적인 키토제닉 다이어트 권장량 기준으로 제시)
2. **자연식품 기반 식단표 구성:** 각 전해질을 효과적으로 섭취할 수 있는 자연식품 위주의 식단표를 하루 3끼 기준으로 작성합니다. 각 식단에는 해당 식품의 1회 제공량과 포함된 전해질 함량을 명시합니다.
3. **식단표 관련 추가 정보 제공:** 식단표에 포함된 식품 외에, 전해질 보충에 도움이 되는 다른 자연식품 목록을 추가적으로 제공합니다.
4. **주의사항 강조:** 과다 섭취 시 발생할 수 있는 부작용과 개인별 건강 상태에 따른 맞춤형 섭취 필요성을 강조합니다.

# Constraints
1. 정보는 최신 연구 결과와 키토제닉 다이어트 가이드라인에 근거해야 합니다.
2. 식단은 현실적으로 실행 가능해야 하며, 구하기 쉬운 자연식품을 중심으로 구성해야 합니다.
3. 정보 제공 시 전문 용어 사용을 최소화하고, 일반인이 이해하기 쉬운 언어를 사용해야 합니다.
4. 식단표는 표 형식으로 깔끔하게 정리하고, 각 식품의 전해질 함량 정보를 정확하게 기재해야 합니다.
5. 어조는 친절하고 조언적인 어조를 유지하며, 독자가 긍정적인 마음으로 식단을 실천할 수 있도록 격려해야 합니다.

# Output Format

다음 형식을 준수하여 작성하세요.

**1. 제목:** 키토 플루 예방을 위한 전해질 섭취 가이드 (키토제닉 다이어트 초반)

**2. 서론:**
* 키토 플루의 원인과 전해질 불균형의 중요성 간략하게 설명
* 키토제닉 다이어트 시 전해질 보충의 필요성 강조

**3. 본론:**

*   **전해질별 하루 권장 섭취량**

    | 전해질   | 하루 권장 섭취량 |
    | ------ | -------- |
    | 나트륨   | [ ] mg   |
    | 마그네슘  | [ ] mg   |
    | 칼륨    | [ ] mg   |
*   **키토 플루 예방 식단표 (예시)**

    | 식사     | 메뉴                                           | 1회 제공량  | 나트륨 (mg) | 마그네슘 (mg) | 칼륨 (mg) |
    | ------ | --------------------------------------------- | -------- | -------- | -------- | -------- |
    | 아침     | [메뉴]                                          | [ ]       | [ ]       | [ ]       | [ ]       |
    | 점심     | [메뉴]                                          | [ ]       | [ ]       | [ ]       | [ ]       |
    | 저녁     | [메뉴]                                          | [ ]       | [ ]       | [ ]       | [ ]       |
*   **전해질 보충에 도움이 되는 다른 자연식품:** (리스트 형식으로 작성)
    *   [식품 1]
    *   [식품 2]
    *   ...

**4. 결론:**

*   전해질 섭취의 중요성 재강조
*   개인별 맞춤형 섭취 필요성 언급 (필요시 전문가 상담 권장)
*   건강한 키토제닉 다이어트를 위한 격려 메시지

**5. 주의사항:** (글머리 기호로 강조)

*   [과다 섭취 시 부작용]
*   [개인별 건강 상태 고려 필요]
*   [기타 주의사항]
```

## 💡 사용 팁

1.  [메뉴] 부분을 사용자의 선호도나 알레르기 유무에 따라 변경하여 맞춤형 식단을 구성할 수 있습니다.
2.  추가적으로 개인의 활동량이나 건강 상태 정보를 제공하면 더욱 정확한 가이드라인을 얻을 수 있습니다.
3.  ChatGPT 또는 Gemini와 같은 대규모 언어 모델에서 효과적입니다.
