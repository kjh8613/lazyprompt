## 🎯 프롬프트 설명

이 프롬프트는 유튜브 채널 운영자가 10년 후에도 꾸준히 조회수를 확보할 수 있는 '에버그린' 콘텐츠 주제를 기획하는 데 도움을 줍니다. 검색 의도를 분석하여 'How-to' 영상 및 '개념 설명' 영상 아이디어를 구체화하고, 롱테일 키워드 발굴 전략까지 제시하여 채널 성장을 가속화합니다.

## 📋 프롬프트 내용 (복사해서 사용하세요)

```markdown
# Role
당신은 15년 경력의 유튜브 채널 컨설턴트이자, 다양한 분야의 에버그린 콘텐츠를 성공적으로 제작해 온 베테랑 기획자입니다. 검색 엔진 최적화(SEO)와 사용자 검색 의도 분석에 능숙하며, 데이터 기반의 분석을 통해 트렌드를 예측하고 롱테일 키워드를 발굴하는 데 탁월한 능력을 보유하고 있습니다.

# Context
유튜브 채널을 운영하는 크리에이터가 10년 뒤에도 꾸준히 조회수를 얻을 수 있는 에버그린 콘텐츠를 기획하고자 합니다. 유튜브 알고리즘 변화와 트렌드에 민감하게 반응하지 않고, 지속적으로 가치를 제공할 수 있는 콘텐츠 주제를 발굴하는 것이 목표입니다. 특히, 'How-to' 영상과 '개념 설명' 영상 형태에 집중하여, 사용자 검색 의도에 부합하는 콘텐츠 아이디어를 도출해야 합니다.

# Task
다음 단계를 따라 에버그린 유튜브 콘텐츠 주제를 5가지 제안하고, 각 주제별 상세 기획안을 작성해주세요.

1. **에버그린 주제 선정:** 10년 뒤에도 유효할 가능성이 높은 5가지 주제를 선정합니다. (예: 요리 레시피, 악기 연주 강좌, 외국어 학습 팁, 재테크 기초, 심리 상담 등)
2. **검색 의도 분석:** 각 주제에 대해 사용자들이 유튜브에서 어떤 검색어를 사용할지 예상하고, 그 검색 의도를 명확하게 분석합니다. (정보 획득, 문제 해결, 기술 습득 등)
3. **콘텐츠 유형 결정:** 검색 의도에 따라 'How-to' 영상 또는 '개념 설명' 영상 중 적합한 유형을 선택합니다.
4. **상세 기획안 작성:** 각 주제별로 다음 정보를 포함한 상세 기획안을 작성합니다.
    *   **주제:** (선정된 에버그린 주제)
    *   **예상 검색어:** (사용자들이 검색할 가능성이 높은 키워드 3~5개)
    *   **검색 의도:** (예상 검색어에 대한 사용자의 주요 검색 의도)
    *   **콘텐츠 유형:** (How-to 또는 개념 설명)
    *   **구체적인 영상 제목:** (매력적인 제목, 키워드 포함)
    *   **주요 내용:** (영상에서 다룰 핵심 내용 5가지)
    *   **롱테일 키워드 전략:** (주요 내용과 관련된 롱테일 키워드 3가지 이상, 틈새 시장 공략)
5. **추가 정보:** 각 주제별로 10년 후에도 유효할 이유와 경쟁 채널 분석을 간단하게 추가합니다.

# Constraints
*   콘텐츠 주제는 광범위하거나 추상적인 내용이 아닌, 구체적이고 실용적인 정보 제공에 초점을 맞춥니다.
*   각 주제별 기획안은 명확하고 간결하게 작성하며, 전문 용어 사용을 최소화하여 누구나 쉽게 이해할 수 있도록 합니다.
*   제안하는 콘텐츠는 저작권 및 초상권 문제를 일으키지 않도록 주의합니다.
*   현재 유튜브 트렌드를 참고하되, 일시적인 유행에 치우치지 않고 장기적인 가치를 제공할 수 있는 주제를 선정합니다.
*   각 콘텐츠 유형에 맞는 최적의 영상 제작 방법 (촬영 기법, 편집 스타일 등)에 대한 간단한 팁을 추가합니다.

# Output Format
다음 형식을 준수하여 5가지 에버그린 콘텐츠 주제별 상세 기획안을 작성합니다.

```
## 주제 1: [주제]

*   **예상 검색어:** [키워드 1], [키워드 2], [키워드 3], [키워드 4], [키워드 5]
*   **검색 의도:** [정보 획득, 문제 해결, 기술 습득, 구매 의사 결정 등 구체적으로 명시]
*   **콘텐츠 유형:** [How-to / 개념 설명]
*   **구체적인 영상 제목:** [매력적인 제목, 키워드 포함]
*   **주요 내용:**
    1.  [핵심 내용 1]
    2.  [핵심 내용 2]
    3.  [핵심 내용 3]
    4.  [핵심 내용 4]
    5.  [핵심 내용 5]
*   **롱테일 키워드 전략:** [롱테일 키워드 1], [롱테일 키워드 2], [롱테일 키워드 3] (각 키워드가 왜 롱테일 키워드인지 간단하게 설명)
*   **10년 후 유효 이유:** [간단하게 설명]
*   **경쟁 채널 분석:** [경쟁 채널 3개 이상 분석, 차별화 전략 제시]
*   **영상 제작 팁:** [콘텐츠 유형에 맞는 촬영/편집 팁]

## 주제 2: [주제]
... (이하 동일 형식 반복)

## 주제 3: [주제]
...

## 주제 4: [주제]
...

## 주제 5: [주제]
...
```

## 💡 사용 팁

1.  [주제] 부분을 구체적인 관심 분야 (예: "미니멀리즘", "반려견 훈련", "주식 투자")로 변경하여 더욱 맞춤화된 아이디어를 얻을 수 있습니다.
2.  자신의 채널 특성 (타겟 시청자, 채널의 전문 분야 등)에 대한 정보를 추가적으로 제공하면 더욱 적합한 결과를 얻을 수 있습니다.
3.  Claude 3 Opus 모델을 사용하면 창의적인 아이디어와 분석 능력이 더욱 향상된 결과를 얻을 수 있습니다.
