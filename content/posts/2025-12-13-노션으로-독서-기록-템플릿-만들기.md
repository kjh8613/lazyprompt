# 노션 독서 기록 템플릿 생성 프롬프트

## 목표:
개인적인 독서 경험을 체계적으로 기록하고 관리할 수 있는 노션 템플릿을 설계합니다.

## 요구사항:

1. **템플릿 구조**:
   - 메인 데이터베이스 (도서 목록)
   - 페이지 템플릿 (각 도서별 상세 정보 기록)
   - 대시보드 (독서 현황 요약)

2. **데이터베이스 속성**:
   - 제목 (필수): 도서 제목
   - 저자 (필수): 도서 저자
   - 읽기 시작일 (날짜): 독서를 시작한 날짜
   - 읽기 완료일 (날짜): 독서를 완료한 날짜
   - 별점 (선택): 1~5점 척도 (다중 선택)
   - 장르 (선택): 소설, 에세이, 자기계발 등 (다중 선택)
   - 상태 (선택): 읽기 전, 읽는 중, 완독 (단일 선택)
   - 키워드 (선택): 책 내용과 관련된 키워드 (다중 선택)
   - 메모 (텍스트): 책에 대한 간단한 메모
   - 인용구 (텍스트): 인상 깊었던 구절
   - 리뷰 (텍스트): 책에 대한 개인적인 리뷰 및 감상
   - 페이지 수 (숫자): 도서의 총 페이지 수
   - 진행률 (수식): (완독 여부에 따라 자동으로 계산되는 진행률, 읽는 중일 경우 수동 입력 가능)
   - 표지 이미지 (파일): 도서 표지 이미지

3. **페이지 템플릿**:
   - 각 도서별 상세 정보 페이지는 위에 정의된 데이터베이스 속성을 모두 포함해야 합니다.
   - '인용구' 섹션: 인상 깊었던 구절을 기록하고, 관련 생각이나 감상을 적을 수 있는 공간을 제공합니다.
   - '리뷰' 섹션: 책 전체에 대한 개인적인 감상, 배운 점, 느낀 점 등을 자유롭게 작성할 수 있는 공간을 제공합니다.
   - '핵심 내용 요약' 섹션: 책의 핵심 내용을 간략하게 요약하여 기록합니다.

4. **대시보드**:
   - 완독한 도서 수 (필터링된 데이터베이스 뷰)
   - 읽고 있는 도서 (필터링된 데이터베이스 뷰)
   - 별점별 도서 목록 (필터링된 데이터베이스 뷰)
   - 장르별 도서 목록 (필터링된 데이터베이스 뷰)
   - 읽기 시작일 기준 정렬된 도서 목록 (캘린더 뷰)

5. **추가 기능**:
   - "다음 책" 버튼: 읽고 싶은 책을 빠르게 추가할 수 있는 버튼 제공
   - 목표 독서량 설정 및 달성률 표시 기능

## 출력 형식:

위 요구사항을 만족하는 노션 템플릿 구조에 대한 자세한 설명과 각 속성 설정 방법을 단계별로 설명해주세요. 필요하다면 노션 데이터베이스 생성 및 활용에 대한 기본적인 정보도 포함해주세요.

## 추가 정보:
- 사용자 레벨: 초급
- 사용 목적: 개인적인 독서 기록 및 관리
- 강조점: 쉽고 직관적인 사용법


## 3. 사용 팁
* **데이터베이스 속성 확장:** 필요한 속성을 추가하여 템플릿을 개인화하세요. 예를 들어 '구매 링크', '관련 도서' 등을 추가할 수 있습니다.
* **뷰 다양화:** 데이터베이스 뷰를 다양하게 활용하여 독서 데이터를 다각도로 분석하세요. 갤러리 뷰를 활용하여 표지 이미지를 한눈에 볼 수도 있습니다.
* **자동화 활용:** 노션의 자동화 기능을 활용하여 독서 기록 프로세스를 더욱 효율적으로 만드세요. 예를 들어 '읽기 완료일'이 입력되면 자동으로 '상태'를 '완독'으로 변경하는 자동화를 설정할 수 있습니다.
