## 1. 개요
이 프롬프트는 **노션으로 독서 기록 템플릿 만들기** 작업을 자동화하거나 퀄리티를 높이고 싶은 분들을 위해 제작되었습니다. 노션 데이터베이스를 효율적으로 활용하여 독서 경험을 체계적으로 기록하고 관리할 수 있도록 돕습니다. 이 프롬프트는 책 제목, 저자, 장르, 읽은 날짜, 별점, 감상, 인상 깊었던 구절 등을 포함한 완벽한 독서 기록 템플릿을 생성하는 데 필요한 모든 요소를 제공합니다.

## 2. 프롬프트 (복사 가능)
아래 박스의 내용을 복사해서 AI(ChatGPT, Midjourney 등)에 붙여넣으세요.

text
**[한글 프롬프트]**

당신은 최고의 노션 전문가입니다. 독서를 즐기는 사람들을 위해 완벽하고 효율적인 노션 독서 기록 템플릿을 디자인하고, 데이터베이스 구조와 각 속성에 대한 자세한 설명을 제공해주세요. 템플릿은 다음 사항을 포함해야 합니다.

1.  **데이터베이스 이름:** "독서 기록"
2.  **속성:**
    *   **책 제목:** (텍스트, 필수)
    *   **저자:** (텍스트, 필수)
    *   **장르:** (선택, 여러 개 선택 가능, 옵션: 소설, 에세이, 자기계발, 과학, 역사, 철학 등)
    *   **읽은 날짜:** (날짜)
    *   **별점:** (선택, 1-5점)
    *   **상태:** (선택, 진행 중, 완독, 중단)
    *   **감상:** (텍스트, 자유롭게 감상을 기록)
    *   **인상 깊었던 구절:** (텍스트, 기억하고 싶은 구절 기록)
    *   **태그:** (선택, 개인적인 태그 추가 가능)
    *   **커버 이미지:** (파일, 책 표지 이미지)
3.  **뷰:**
    *   **전체 보기:** (테이블, 모든 속성 표시)
    *   **갤러리 보기:** (갤러리, 커버 이미지와 책 제목 표시, 필요에 따라 다른 속성 추가)
    *   **장르별 보기:** (보드, 장르별 그룹화)
    *   **캘린더 보기:** (캘린더, 읽은 날짜 표시)
4.  **추가 기능:**
    *   **템플릿 버튼:** 새 책 기록을 빠르게 추가할 수 있는 템플릿 버튼 생성
    *   **관계형 데이터베이스 연결:** 다른 데이터베이스(예: "작가 정보")와 연결하여 더 자세한 정보 관리 가능 (선택 사항)

**[영어 프롬프트]**

You are a top Notion expert. Design a perfect and efficient Notion Reading Log template for book lovers, providing detailed explanations of the database structure and each property. The template should include the following:

1.  **Database Name:** "Reading Log"
2.  **Properties:**
    *   **Book Title:** (Text, Required)
    *   **Author:** (Text, Required)
    *   **Genre:** (Select, Multiple Select Allowed, Options: Fiction, Essay, Self-Help, Science, History, Philosophy, etc.)
    *   **Date Read:** (Date)
    *   **Rating:** (Select, 1-5 stars)
    *   **Status:** (Select, In Progress, Completed, Abandoned)
    *   **Thoughts:** (Text, Free to record thoughts)
    *   **Memorable Quotes:** (Text, Record memorable quotes)
    *   **Tags:** (Select, Add personal tags)
    *   **Cover Image:** (File, Book cover image)
3.  **Views:**
    *   **All View:** (Table, Show all properties)
    *   **Gallery View:** (Gallery, Show cover image and book title, add other properties as needed)
    *   **Genre View:** (Board, Grouped by genre)
    *   **Calendar View:** (Calendar, Show date read)
4.  **Additional Features:**
    *   **Template Button:** Create a template button to quickly add new book entries.
    *   **Relational Database Connection:** Connect to another database (e.g., "Author Information") for more detailed information management (Optional)


## 3. 사용 팁
*   **구체적인 요구사항 추가:** 프롬프트에 원하는 속성이나 뷰를 구체적으로 명시하면 더욱 맞춤화된 템플릿을 얻을 수 있습니다. 예를 들어, "독서 목표" 속성을 추가하거나, "월별 독서량" 요약 뷰를 추가해달라고 요청할 수 있습니다.
*   **예시 데이터 제공:** AI에게 템플릿에 사용할 예시 데이터를 제공하면 템플릿 디자인의 방향성을 제시할 수 있습니다. 예를 들어, "책 제목: 해리포터와 마법사의 돌, 저자: J.K. 롤링, 장르: 판타지"와 같은 정보를 제공하면 됩니다.
*   **다양한 스타일 시도:** 프롬프트의 톤앤매너를 바꿔보세요. "미니멀한 디자인"이나 "화려한 디자인"과 같이 원하는 스타일을 명시하면 AI가 다양한 스타일의 템플릿을 제안해 줄 것입니다.
