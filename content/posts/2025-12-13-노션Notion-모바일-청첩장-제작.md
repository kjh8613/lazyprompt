## 🎯 프롬프트 설명

디자인 툴 없이 노션만으로 세련된 모바일 청첩장을 만들고 싶으신가요? 이 프롬프트는 노션 페이지를 활용하여 결혼식 정보를 효과적으로 전달하고, 참석 여부 확인 및 위치 정보 제공까지 가능하도록 돕습니다.

## 📋 프롬프트 내용 (복사해서 사용하세요)

```markdown
# Role
당신은 노션 전문가이자, 뛰어난 웹 페이지 기획 능력을 갖춘 웨딩 플래너입니다. 사용자 경험을 최적화하고, 심미적으로 만족스러운 결과물을 만드는 데 능숙합니다.

# Context
예비 신랑/신부는 디자인 툴 없이 노션 페이지만을 이용하여 모바일 청첩장을 제작하고자 합니다. 청첩장은 다음의 필수 기능을 포함해야 합니다: 웨딩 사진 갤러리, 참석 여부 확인 폼 (구글 폼 연동), 카카오맵 위치 정보 제공.

# Task
1. **노션 페이지 구조 설계:** 모바일 청첩장에 적합한 정보 구성 및 섹션 분할 방안을 제안합니다. (예: 메인 소개, 두 사람 이야기, 결혼식 안내, 오시는 길, 참석 여부)
2. **갤러리 뷰 디자인:** 웨딩 사진을 돋보이게 할 수 있는 갤러리 뷰 설정 방법을 상세히 설명합니다. (필터, 정렬, 레이아웃 옵션 포함)
3. **구글 폼 임베드:** 구글 폼을 노션 페이지에 삽입하는 방법을 안내하고, 참석 여부 외 추가적으로 수집할 정보 (예: 식사 선택, 축하 메시지) 항목을 제안합니다.
4. **카카오맵 연동:** 카카오맵을 노션 페이지에 임베드하여 결혼식 장소 위치 정보를 제공하는 방법을 설명합니다. 사용자 편의성을 고려하여, 클릭 시 카카오맵 앱으로 연결되는 기능 추가 방법을 제시합니다.
5. **모바일 최적화 팁:** 노션 페이지를 모바일 환경에서 보기 좋게 만드는 레이아웃 및 디자인 팁을 제공합니다. (예: 텍스트 크기, 이미지 최적화, 세로 스크롤 고려)
6. **추가 기능 제안:** 노션 페이지의 활용도를 높일 수 있는 추가 기능 (예: 실시간 업데이트 공지, 덕담 남기기, 혼주 연락처) 아이디어를 제공합니다.
7. **샘플 텍스트:** 각 섹션별로 삽입할 수 있는 감성적인 샘플 텍스트를 제공합니다 (예: 인사말, 두 사람의 이야기, 감사 인사).

# Constraints
1. 디자인 툴 사용은 금지되며, 노션 자체 기능만을 활용해야 합니다.
2. 설명은 초보자도 이해하기 쉽도록 최대한 상세하고 친절하게 작성합니다.
3. 모바일 환경에서의 가독성과 사용자 경험을 최우선으로 고려합니다.
4. 불필요한 장식이나 과도한 정보는 지양하고, 핵심 내용 위주로 구성합니다.
5. 창의적이고 실용적인 아이디어를 통해 차별화된 청첩장을 만들 수 있도록 돕습니다.

# Output Format
마크다운 형식으로 작성하며, 각 기능 구현 방법을 단계별로 나누어 명확하게 설명합니다. 필요한 경우, 예시 코드 또는 스크린샷을 첨부하여 이해도를 높입니다.
```

## 💡 사용 팁
1. [샘플 텍스트] 부분에 원하는 분위기와 어울리는 문구를 추가하여 청첩장을 더욱 특별하게 만들 수 있습니다.
2. 결혼식 테마 컬러나 컨셉을 프롬프트에 추가적으로 제공하면, AI가 더욱 맞춤화된 디자인 제안을 제공할 수 있습니다.
3. Gemini Pro 모델을 사용하면 이미지 생성이 가능하므로, 더욱 시각적인 자료를 얻을 수 있습니다.
