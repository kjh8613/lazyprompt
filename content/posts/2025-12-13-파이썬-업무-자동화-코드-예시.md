## 1. 개요
이 프롬프트는 **파이썬 업무 자동화 코드 예시** 작업을 자동화하거나 퀄리티를 높이고 싶은 분들을 위해 제작되었습니다. 예를 들어, 특정 파일들을 찾아서 이름을 바꾸거나, 엑셀 파일을 읽어 데이터를 처리하거나, 웹 페이지를 스크래핑하는 코드 등을 생성하는 데 도움을 받을 수 있습니다.

## 2. 프롬프트 (복사 가능)
아래 박스의 내용을 복사해서 AI(ChatGPT, Midjourney 등)에 붙여넣으세요.

text
다음 설명을 기반으로 파이썬 코드를 작성해주세요. 코드에는 주석을 상세하게 달아 이해를 돕고, 발생 가능한 예외 상황에 대한 처리도 포함해주세요. 또한, 코드를 실행하기 위한 필요한 라이브러리를 `pip install` 명령어를 사용하여 알려주세요.

1. **작업 목표:** [여기에 원하는 자동화 작업을 구체적으로 설명하세요. 예: 특정 폴더 내의 모든 `.txt` 파일을 찾아 'YYYYMMDD_파일명.txt' 형식으로 이름을 변경하고, 변경된 파일명들을 `renamed_files.txt` 파일에 저장합니다.]

2. **입력:** [작업에 필요한 입력 정보를 구체적으로 설명하세요. 예: 대상 폴더의 경로, 변경할 파일 확장자, 변경할 파일명의 형식 등]

3. **출력:** [생성해야 할 출력 정보 또는 결과물을 구체적으로 설명하세요. 예: 파일 이름이 변경된 파일들, 생성된 `renamed_files.txt` 파일, 화면에 출력되는 진행 상황 메시지 등]

4. **예외 처리:** [발생 가능한 예외 상황과 그에 대한 처리 방법을 구체적으로 설명하세요. 예: 대상 폴더가 존재하지 않는 경우, 파일 접근 권한이 없는 경우, 파일 이름 변경에 실패하는 경우 등]

5. **추가 요구 사항:** [코드에 추가적으로 필요한 요구 사항을 구체적으로 설명하세요. 예: 진행률 표시, 로그 파일 생성, 특정 조건에 따른 파일 필터링 등]

예시:

"작업 목표: 지정된 디렉토리 내의 모든 엑셀 파일 (.xlsx)을 읽고, 각 파일의 첫 번째 시트에서 '이름' 열과 '이메일' 열의 데이터를 추출하여 '이름_이메일.txt' 형식의 텍스트 파일로 저장합니다. 각 텍스트 파일에는 해당 엑셀 파일의 이름이 헤더로 포함되어야 합니다.

입력: 엑셀 파일이 있는 디렉토리 경로.

출력: 추출된 데이터를 담은 텍스트 파일들. 각 파일명은 '이름_이메일.txt' 형식이며, 엑셀 파일명은 텍스트 파일의 헤더로 사용됩니다.

예외 처리: 디렉토리가 존재하지 않는 경우, 엑셀 파일이 손상된 경우, '이름' 또는 '이메일' 열이 엑셀 파일에 없는 경우를 처리합니다.

추가 요구 사항:  각 파일이 처리될 때마다 콘솔에 진행률을 표시합니다. 오류가 발생한 파일의 목록을 'error_log.txt' 파일에 기록합니다."

이제, 위의 가이드라인에 따라 원하는 작업을 구체적으로 설명하여 파이썬 코드를 생성하도록 요청하세요.


## 3. 사용 팁
* **구체적인 설명:** 작업 목표, 입력, 출력, 예외 처리 등을 최대한 구체적으로 설명할수록 더 정확하고 원하는 코드를 얻을 수 있습니다.
* **예시 활용:** 제공된 예시를 참고하여 자신의 상황에 맞게 수정하면 더욱 효과적인 프롬프트를 만들 수 있습니다.
* **단계별 요청:** 한 번에 복잡한 코드를 요청하기보다는, 간단한 기능부터 구현하고 점진적으로 기능을 추가하는 방식으로 접근하면 오류를 줄일 수 있습니다.
