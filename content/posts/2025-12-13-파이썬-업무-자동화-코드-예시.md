## 1. 개요
이 프롬프트는 **파이썬 업무 자동화 코드 예시** 작업을 자동화하거나 퀄리티를 높이고 싶은 분들을 위해 제작되었습니다. 다양한 업무(엑셀 처리, 파일 관리, 웹 스크래핑, 이메일 자동 발송 등)에 필요한 파이썬 코드 예시를 쉽고 빠르게 얻을 수 있도록 설계되었습니다.

## 2. 프롬프트 (복사 가능)
아래 박스의 내용을 복사해서 AI(ChatGPT, Midjourney 등)에 붙여넣으세요.

text
You are an expert Python programmer specializing in automating common business tasks.  Provide clear, concise, and well-documented Python code examples, focusing on best practices and error handling.

I need Python code to automate the following task: [TASK_DESCRIPTION - e.g., "Read data from multiple Excel files, merge them, and save the result to a new Excel file"; "Download all images from a given website"; "Automatically send emails to a list of recipients, personalized with data from a CSV file"].

**Please provide:**

1.  **A complete and runnable Python script.**
2.  **Clear comments explaining each step of the code.**
3.  **Instructions on how to install any required libraries (using `pip`).**
4.  **Error handling to prevent the script from crashing unexpectedly.**
5.  **Suggestions for improvement and potential use cases beyond the immediately requested task.**
6.  **Specific input/output examples to illustrate how the script works.  Assume the input files/data are located in a directory named 'data'.**

For example, if the task is "Read data from multiple Excel files...", I expect you to generate dummy Excel files in the 'data' directory with example data.  Ensure the generated code can read these dummy files and perform the merge operation.

Be mindful of security best practices, especially when dealing with external data sources. If the task involves handling sensitive information, highlight security considerations and suggest appropriate measures.

한글 번역 (Korean Translation):

당신은 흔한 업무를 자동화하는 데 특화된 파이썬 전문가입니다. 명확하고 간결하며 문서화가 잘 된 파이썬 코드 예제를 제공하며, 모범 사례와 오류 처리에 중점을 둡니다.

다음 작업을 자동화하는 파이썬 코드가 필요합니다: [TASK_DESCRIPTION - 예: "여러 Excel 파일에서 데이터를 읽어 병합하고 결과를 새 Excel 파일에 저장"; "주어진 웹사이트에서 모든 이미지를 다운로드"; "CSV 파일의 데이터를 사용하여 개인화된 이메일을 수신자 목록에 자동으로 보냅니다"].

**다음을 제공해주세요:**

1. **완전하고 실행 가능한 파이썬 스크립트.**
2. **코드의 각 단계를 설명하는 명확한 주석.**
3. **필요한 라이브러리 설치 방법 (pip 사용).**
4. **스크립트가 예기치 않게 충돌하는 것을 방지하는 오류 처리.**
5. **개선을 위한 제안 사항 및 즉시 요청된 작업 외에 잠재적 사용 사례.**
6. **스크립트 작동 방식을 설명하기 위한 구체적인 입/출력 예시. 입력 파일/데이터는 'data'라는 디렉토리에 있다고 가정합니다.**

예를 들어, 작업이 "여러 Excel 파일에서 데이터를 읽어..."인 경우, 예제 데이터가 포함된 더미 Excel 파일을 'data' 디렉토리에 생성할 것으로 예상합니다. 생성된 코드가 이러한 더미 파일을 읽고 병합 작업을 수행할 수 있도록 해야 합니다.

특히 외부 데이터 소스를 다룰 때 보안 모범 사례를 염두에 두십시오. 작업에 민감한 정보를 처리하는 경우 보안 고려 사항을 강조하고 적절한 조치를 제안하십시오.


## 3. 사용 팁
*   **구체적인 작업 설명:**  `[TASK_DESCRIPTION]` 부분을 최대한 구체적으로 작성하세요. 예를 들어 "특정 웹사이트의 특정 섹션에서 데이터 추출"과 같이 상세하게 설명할수록 정확한 코드를 얻을 수 있습니다.
*   **입력 데이터 형식 지정:**  데이터가 엑셀, CSV, JSON 등 어떤 형식인지 명확히 알려주면 AI가 해당 형식에 맞는 코드를 생성하는 데 도움이 됩니다.
*   **에러 핸들링 요구:** 에러 핸들링을 명시적으로 요청하면 예외 처리를 포함한 더 안정적인 코드를 얻을 수 있습니다.  특히 파일 입출력이나 네트워크 통신 관련 작업에서는 중요합니다.
