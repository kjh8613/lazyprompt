## 1. 개요

## 2. 프롬프트 (복사 가능)

### Prompt Start ###

**[Korean]**
스테이블 디퓨전에서 사용할 3D 캐릭터 모델을 추천해주세요. 다음과 같은 스타일과 설정을 고려해주세요:

*   **스타일:**
    *   리얼리스틱 (Realistic)
    *   만화 (Cartoon)
    *   로우 폴리 (Low Poly)
    *   사이버펑크 (Cyberpunk)
    *   판타지 (Fantasy)

*   **포즈:**
    *   서 있는 자세 (Standing)
    *   걷는 자세 (Walking)
    *   앉아 있는 자세 (Sitting)
    *   액션 포즈 (Action Pose, 예: 달리기, 점프)

*   **디테일:**
    *   텍스처 퀄리티 (높음, 중간, 낮음)
    *   폴리곤 수 (높음, 중간, 낮음)
    *   라이팅 설정 (예: 앰비언트 오클루전, 그림자)

*   **추가 설정:**
    *   애니메이션 가능 여부 (가능, 불가능)
    *   리그 설정 여부 (리그 있음, 리그 없음)
    *   파일 형식 (FBX, OBJ, GLTF)

각 스타일별로 적합한 3D 모델 및 관련 프롬프트를 제안해주세요. 모델의 특징과 스테이블 디퓨전에서 효과적으로 사용하기 위한 팁도 함께 알려주세요.

**[English]**
Recommend 3D character models for Stable Diffusion, considering the following styles and settings:

*   **Style:**
    *   Realistic
    *   Cartoon
    *   Low Poly
    *   Cyberpunk
    *   Fantasy

*   **Pose:**
    *   Standing
    *   Walking
    *   Sitting
    *   Action Pose (e.g., Running, Jumping)

*   **Details:**
    *   Texture Quality (High, Medium, Low)
    *   Polygon Count (High, Medium, Low)
    *   Lighting Settings (e.g., Ambient Occlusion, Shadows)

*   **Additional Settings:**
    *   Animatable (Yes, No)
    *   Rigged (Rigged, Unrigged)
    *   File Format (FBX, OBJ, GLTF)

Suggest suitable 3D models and related prompts for each style. Provide information about the model's characteristics and tips for effective use in Stable Diffusion.

### Prompt End ###


## 3. 사용 팁
*   **스타일 가이드 활용:** 원하는 스타일(리얼리스틱, 만화 등)을 구체적으로 지정하여 결과물의 분위기를 조절하세요. 예를 들어, "Realistic, hyper-detailed, 8k"와 같은 키워드를 추가하면 더욱 사실적인 결과물을 얻을 수 있습니다.
*   **포즈 및 액션 강조:** 캐릭터의 포즈는 이미지의 전체적인 느낌을 크게 좌우합니다. "Dynamic pose, action shot, vibrant colors"와 같이 역동적인 포즈와 함께 시각적인 요소를 강조하면 더욱 흥미로운 이미지를 만들 수 있습니다.
*   **세부 묘사 추가:** 모델의 디테일(텍스처, 폴리곤 수 등)은 결과물의 품질에 직접적인 영향을 미칩니다. "High-quality textures, intricate details, cinematic lighting"과 같은 구체적인 묘사를 추가하여 디테일을 살릴 수 있습니다.
